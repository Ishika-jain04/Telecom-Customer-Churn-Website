# Telecom Customer Churn Website

## Overview

The Telecom Customer Churn Website is a modern, responsive web application designed to analyze customer information and provide meaningful insights into customer churn. The application offers an intuitive user interface, interactive dashboards, and a scalable architecture to deliver an efficient and seamless user experience.

---

## Features

- Responsive and user-friendly interface
- Customer data analysis
- Churn prediction interface
- Interactive dashboards
- Modern and clean UI design
- Optimized performance
- Scalable project architecture
- Clean and maintainable codebase

---

- Vite
- HTML5
- CSS3

- Tailwind CSS

## Installation

### Clone the repository

```bash
git clone https://github.com/ishika-jain04/Telecom-Customer-Churn-Website.git
```

### Navigate to the project directory

```bash
cd Telecom-Customer-Churn-Website
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
